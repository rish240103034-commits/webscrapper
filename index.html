Deployment Guide

Free stack: Render + GitHub Pages + xAI Grok

1
Get xAI API Key
→ console.x.ai → Sign up → API Keys → Create
→ Copy key starting with xai-...
→ Enable Data Sharing for $150/month free credits
2
Deploy Backend on Render
→ render.com → Sign up with GitHub
→ New → Web Service → connect your repo
→ Start command: node server.js → Free tier
→ Add env var: XAI_API_KEY = your key
→ Copy your URL: https://yourapp.onrender.com
3
Deploy Frontend on GitHub Pages
→ Upload files to GitHub repo
→ Settings → Pages → Branch: main → root /
→ Live at: https://username.github.io/repo
4
Configure & Scrape
→ Paste your Render URL in the Backend URL field
→ Pick a preset or enter any URL
→ Click Run Scraper 🚀

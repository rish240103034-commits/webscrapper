Free Deployment Guide

Full stack: xAI Grok API + Render.com + GitHub Pages. Zero cost for most dev workloads.

New xAI accounts get $25 free credits on signup + $150/month via the data-sharing program at console.x.ai
1 Get your xAI API Key
→ Go to console.x.ai
→ Create account → API Keys → New Key
→ Copy the key (starts with xai-)
→ Optionally: Settings → Data Sharing → enable for $150/mo free credits
2 Deploy backend to Netlify (free)
→ Download this project's zip (GitHub repo)
→ app.render.com → "Add new site" → Deploy manually
→ Or: netlify init → link GitHub repo → auto-deploy
→ Site settings → Environment variables → Add XAI_API_KEY
→ Your backend URL: https://yoursite.onrender.com/scrape
3 Deploy frontend to GitHub Pages (free)
→ Push public/index.html to your GitHub repo
→ Settings → Pages → Source: main branch → /public
→ Your URL: https://username.github.io/repo-name
→ Or use Netlify to host both (simpler)
4 Configure and scrape
→ Paste your API key in the left panel
→ Paste your Netlify backend URL
→ Enter target URL, set schema, click run
→ Export as JSON or CSV

Deployment Guide

Free stack: Render.com backend + GitHub Pages frontend. No API keys needed.

1
Push code to GitHub
Upload server.js, index.html, package.json to a new public repo
2
Deploy backend on Render
→ render.com → New → Web Service
→ Connect repo → Start: node server.js → Free tier → Deploy
3
Enable GitHub Pages
→ Repo → Settings → Pages → Branch: main → / root → Save
4
Update backend URL
In index.html update BACKEND_URL to your Render URL and commit
